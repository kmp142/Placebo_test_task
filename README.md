# Функционал
* При нажатии на аватар открывается UIImagePickerController, в котором вы можете выбрать фото из своей галереи
* При нажатии на диаграмму происходит обновление данных с анимацией, данные генерируются рандомно
* При обновлении данных на экране редактирования - имени и возраста, а также при изменении фото сохранение данных происходит в UserDefaults.
